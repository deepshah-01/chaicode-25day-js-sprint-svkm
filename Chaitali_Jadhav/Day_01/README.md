## Video 2: Local JavaScript Environment Setup

### Tools Required
- Node.js
- VS Code

### Installation Steps

#### 1. Install Node.js
- Visit https://nodejs.org
- Download the LTS version.
- Run the installer and complete the setup.

#### 2. Verify Installation
Open Terminal/Command Prompt and run:

```bash
node -v
```

If a version number appears, Node.js is installed successfully.

#### 3. Install VS Code
- Download VS Code.
- Install and launch it.
- Create a project folder.

#### 4. Create First JavaScript File

Create a file named:

```text
text.js
```

Add:

```javascript
console.log("Hello World");
```

#### 5. Run the Program

Open terminal in VS Code and run:

```bash
node text.js
```

Output:

```text
Hello World
```

## Video 3: GitHub Codespaces Setup

### Steps

1. Create a GitHub account.
2. Create a new repository.
3. Click **Code → Codespaces → Create Codespace**.
4. Wait for the online VS Code environment to load.
5. Configure Node.js environment.
6. Create `text.js`.
7. Run:

```bash
node text.js
```

8. Save changes and commit them to GitHub.
