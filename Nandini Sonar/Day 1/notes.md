```js

```

When I ran it successfully, it felt amazing because this was my first JavaScript program.

---

## Things I Learned

### JavaScript Files

- JavaScript files use the `.js` extension.
- Unlike normal text files, JavaScript files contain executable code.
- The browser and Node.js can understand and execute these files.

### JavaScript Beyond Browsers

Earlier, JavaScript was mostly used inside browsers.

Now it is used for:

✅ Frontend Development

✅ Backend Development

✅ Mobile Applications

✅ Full Stack Development

✅ APIs and Server Applications

---

## JavaScript Runtime Environment

### Node.js

- Allows JavaScript to run outside the browser.
- Helps execute JavaScript files directly from the terminal.

### Deno

- Another JavaScript runtime environment.
- Created by Ryan Dahl (the creator of Node.js).

---

## Code Editors

Some popular code editors are:

- VS Code
- JetBrains Fleet
- Sublime Text

### My Choice

VS Code because:

- Beginner friendly
- Free to use
- Most widely used in industry

---

## Running JavaScript

Check Node.js version:

```bash
node -v
```

Run JavaScript file:

```bash
node test.js
```

Example:

```js
console.log("Nandini Sonar");
```

# 🌐 Lecture 3: GitHub Codespaces / Workspace

Today I also learned how to code directly in the cloud using GitHub Codespaces.

It felt like VS Code running inside the browser.

---

## What I Learned

- GitHub can store all my code online.
- Codespaces provides a complete development environment.
- I can write, edit, run, and manage code without installing everything locally.
- It is useful when my local setup is not working properly.
